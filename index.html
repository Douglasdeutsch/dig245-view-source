DIG 245 View Source

How to view source code of a web page in firefox

  1. Pull up the website you want to analyze
  2. Right click anywhere on the page and click inspect

3 other things you can do with dev tools

  • Dev tools can debug your website
  • Dev tools can be used to apply styles
  • You can make temporary changes in the html

Looking at all of these features of a website is like 'pulling back the curtain of a website'
